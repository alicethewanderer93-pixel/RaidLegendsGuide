# Images Folder

Add character images here with these filenames:

- **arbiter.jpg** - Arbiter portrait
- **siphi.jpg** - Siphi the Lost Bride portrait  
- **rotos.jpg** - Rotos the Lost Groom portrait
- **mithrala.jpg** - Mithrala Lifebringer portrait
- **miscreated-monster.jpg** - Miscreated Monster portrait
- **frozen-baneling.jpg** - Frozen Baneling portrait

## Instructions:

1. Save character images as JPG files
2. Rename them to match the filenames above
3. Upload them to this `images/` folder
4. The app will automatically display them next to each champion

## Recommended Image Specs:

- **Size:** 200x200px or larger (will scale automatically)
- **Format:** JPG or PNG
- **Quality:** High quality images for best appearance

The HTML is set to gracefully hide missing images, so you can add them gradually!
